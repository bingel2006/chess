双人对战象棋游戏 - 完整游戏规则
游戏规则
1. 准备阶段

双方轮流购买棋子（最多9个 + 1国王自动添加）。
初始财富：1500💰。
棋子类型：士兵、谋士、将军、工人、刺客。
选择 HP 等级，价格越高 HP 越高（升级选项见购买界面）。
完成选择后，棋子随机放置在后排（避免障碍）。
国王初始 HP=150, ATK=30, 速度=2（金色标识）。

2. 游戏阶段

回合制：玩家1 先手，轮流行动。结束回合切换玩家。
行动顺序（本回合限1次主要行动）：

移动/攻击：点击你的棋子选择（黄色高亮），然后点击曼哈顿距离 <= 速度 的格子。

空位：移动到该格（可能触发障碍 ±30 HP）。
敌方：攻击！互损（双方 HP -= 对方 ATK）。防御方死亡后，攻击方移至其位置。
河格（第9列）：不可移动/占领。

拆分棋子：选棋子后点击按钮，随机拆分成2-4个子单位（HP 均分，需周边空位）。视为本回合行动。
组队攻击：切换组队模式，点击相邻你的棋子选组（橙色高亮，≥2个），点击相邻敌方执行。总 ATK vs 目标 HP：成功击败目标（攻击方 -10 HP），失败全死。
建造堡垒：切换堡垒模式，选择类型（2倍原价），点击空置格子放置。堡垒不可移动（速度=0），可攻击相邻敌方；被攻击伤害减半。视为本回合行动。
建造城墙：切换墙模式，点击空位放置（50💰），阻挡敌方通过，但不能完全闭合（相邻墙≥3不可建）。
建造补给站：切换补给模式，点击空位放置（100💰），己方棋子进入该格满血恢复。
建造农田：切换农田模式，点击空位放置（50💰），每回合结束生产50💰，HP:20，被攻击后归属攻击方。
工人特殊：可在城墙旁（相邻）拆除敌方城墙，耗费如攻击棋子（互损）。
刺客特殊：攻击伤害为ATK的0.5/1/2倍（随机，等概率）。
谋士特殊：对同一敌方棋子攻击2次后，该棋子叛变加入己方（外形变为己方形状）。

随机事件：每玩家3次抽取（回合内用，视为行动）。抽取后若需目标，点击任意棋子应用。

陨石攻击：-50 HP 给选棋子。
发现石油：+200💰 财富。
神秘治疗：选棋子满 HP。
地震：所有棋子 -10 HP。
幸运buff：选棋子 +20 ATK（本回合）。

4. 胜利与注意

胜利：击败敌国王（播放音效，弹窗宣布）。
障碍：陆地随机 +30(绿)/-30(红) HP，首次踩触发。
UI：棋子显示 HP/总HP，点击查看详情（底部面板）。
提示：行动后高亮消失。无行动可结束回合。
重置：点击“重新开始”从头。
技巧：保护国王，用高 ATK 围攻敌王；事件/组队放大优势；堡垒建前线阻敌。
